# **Multi-Platform Identity, Access, & Systems Administration Portfolio**

This repository contains documentation and architectural overviews for four comprehensive projects demonstrating identity governance, role-based access control (RBAC), and cloud infrastructure setup. These simulations reflect production-ready environments designed to scale while maintaining strict security boundaries.

**📁 Portfolio Projects**

1. **Google Workspace** - User Lifecycle Management

Core Focus: Automated onboarding/offboarding, directory segmentation, and application-level service overrides.

Key Outcome: Created a functional hierarchy that isolates department data and enforces domain-wide software governance policies.

2. **Microsoft Entra ID** - Identity Governance & Session Control

Core Focus: Emergency recovery management, bulk provisioning automation, and active threat response mitigation.

Key Outcome: Configured break-glass infrastructure parameters and global token revocations alongside targeted multi-factor authentication (MFA) rollouts.

3. **QuickBooks Online** - Financial Infrastructure & Governance

Core Focus: Ledger environment deployment, bulk inventory/client ingestion, and data security mapping.

Key Outcome: Established a standardized chart of accounts and customized billing workflows while insulating sensitive banking layers from unauthorized edits.

4. **Zoho Books** - Cloud Accounting & Application Administration

Core Focus: Custom RBAC engineering, multi-tiered localized taxation mapping, and master database provisioning.

Key Outcome: Created distinct functional roles for junior bookkeepers and deployed automated GST grouping matrices to ensure compliant transaction processing.
